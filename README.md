# Smart Airport – AI-Powered Flight Delay Prediction & Dashboard

![Project Screenshot](screenshots/dashboard_home.png)

## Project Overview
This project is an **interactive airport operations dashboard** that leverages **Python** and **Power BI** to predict and visualize flight delays, passenger flow, baggage handling, and security queues.

The solution provides actionable insights for airport operations teams to optimize gate allocation, improve passenger experience, and minimize flight delays.

---

## Key Features
- **Flight Delay Prediction**: Predicts flight delays and their risk levels using a machine learning model based on airline, terminal, day, and weather.
- **Interactive Dashboard**:
  - Visualizes departure delays, average delay by airline and gate.
  - Tracks passenger flow and security queues.
  - AI-powered insights using Power BI Smart Narratives.
- **Data Sources**: Cleaned datasets:
  - `flights_clean.csv`
  - `baggage_clean.csv`
  - `passengers_clean.csv`
  - `passenger_flow_clean.csv`
- **Technical Stack**: Python (Dash, Plotly, Pandas, Scikit-learn), Power BI, and ML for predictive analytics.
